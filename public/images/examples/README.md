# Work Examples Directory

This directory contains actual work examples for each service category.

## Structure:
```
examples/
├── impresion/          # Digital printing examples
├── laser/              # Laser cutting/engraving examples  
└── papeleria/          # Stationery and stickers examples
```

## Naming Convention:
Use descriptive names following the pattern: `service_description.example.webp`

Examples:
- `flyer_restaurant.example.webp`
- `business_cards_lawyer.example.webp`
- `wood_sign_cafe.example.webp`
- `metal_plaque_award.example.webp`
- `trifold_corporate.example.webp`
- `stickers_brand_logo.example.webp`

## Image Requirements:
- Format: WebP (preferred) or JPG
- Resolution: Minimum 800x600px
- Quality: High quality for portfolio display
- File size: Optimized for web (under 500KB recommended)
